## Что такое Git

Git — это система контроля версий, которая позволяет отслеживать изменения в коде и работать над проектом вместе с другими разработчиками.

## Зачем нужен Git

Git помогает:

- сохранять историю изменений
- возвращаться к предыдущим версиям
- работать в команде
- не терять код

## Основные команды

- `git init` — создать репозиторий  
- `git add` — добавить файлы  
- `git commit` — сохранить изменения  
- `git push` — отправить на GitHub  

## Вывод

Git является важным инструментом для любого программиста и используется практически во всех проектах.
